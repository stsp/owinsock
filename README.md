# Open Winsock
winsock-1.1 for openwatcom

Written with the use of [Windows Sockets specification](
https://www.ibiblio.org/pub/micro/pc-stuff/ms-windows/winsock/winsock-1.1/winsock.html)

Uses [libd2sock](https://github.com/stsp/libd2sock)
for [dosemu2](https://github.com/dosemu2/dosemu2) compatibility.
